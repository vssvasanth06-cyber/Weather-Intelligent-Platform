# Weather Intelligent Platform

IoT-based Weather Monitoring Platform

## Features

- Temperature Monitoring
- Humidity Monitoring
- Wind Speed Monitoring
- Rainfall Monitoring
- Dashboard Visualization
- Historical Data Analysis
- Real-time Monitoring

## Technology Stack

Frontend:
- React
- TypeScript
- Tailwind CSS

Backend:
- Python
- FastAPI

Database:
- PostgreSQL

## Installation

Frontend:

npm install
npm run dev

Backend:

pip install -r requirements.txt
uvicorn main:app --reload
